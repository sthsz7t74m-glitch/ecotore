# エコトレ（仮称）30問試作版

環境社会を10分野から学ぶ、スマートフォン中心の非公式学習アプリです。

## 試作版でできること

- 10章の要点表示
- 各章3問、合計30問
- 単一選択、複数選択、○×、穴埋め、用語対応
- 1問ごとの詳細解説
- 10問ミニ模試
- 誤答・要復習・復習期限の保存
- 総合・分野別正答率
- 今日の問題の自動選択
- 学習履歴の書き出し・読み込み・リセット

## 起動方法

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開きます。

## ビルド

```bash
npm run build
```

`dist` フォルダにGitHub Pagesへ配置できる静的ファイルが生成されます。

## GitHub Pages

`vite.config.ts` の `base` は相対パスに設定済みです。GitHub Actionsなどで `dist` をPagesへデプロイできます。
