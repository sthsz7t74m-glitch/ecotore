# エコトレ 完成版 v1.0.0

環境社会を10分野から体系的に学べる、スマートフォン中心の非公式Web学習アプリです。

## 収録内容

- 10章・104重要概念・全520問
- 単一選択、複数選択、○×、穴埋め、用語対応
- 要点教材、3段階の図解、章別確認問題
- 正答理由、選択肢別解説、関連用語、覚え方、一次情報
- 基礎、標準、応用、時事の4段階

## 学習機能

- 今日の問題を復習期限・未回答・苦手分野から自動選択
- 分野、難易度、形式、問題数を指定した演習
- 100問・90分の本番想定模試、10問・30問・苦手・時事模試
- 誤答、要復習、間隔反復、分野別弱点分析
- 総合・分野別正答率、連続学習、累計学習日
- 学習履歴のJSON書出し・読込み

## 運用機能

- PWA・ホーム画面追加
- 一度開いた問題・画面のオフライン利用
- 新版公開時の自動更新
- 問題エディター
- CSV一括取込・書出し
- 問題の新規作成、編集、複製、削除、検証、プレビュー
- 公開用JSON出力
- 右上のバージョン表示

## 起動

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

`dist` にGitHub Pagesへ配置できる静的ファイルが生成されます。`main` ブランチへpushすると、同梱のGitHub Actionsが自動でビルド・公開します。

## 注意

- 本アプリは非公式であり、東京商工会議所その他の団体による公認・提携サービスではありません。
- 公式テキストや過去問題は転載せず、公的機関・国際機関の公開情報を基に独自作成しています。
